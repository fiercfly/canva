# Canva Clone Project

This is a simple web application built using React, inspired by some of the features of Canva.

To run this project on your local machine, follow these steps:

### 1. Prerequisites

You must have [Node.js](https://nodejs.org/) (which includes npm) installed on your computer.

### 2. Installation

Clone this repository and navigate into the project directory:

```bash
git clone [https://github.com/fiercfly/canva.git](https://github.com/fiercfly/canva.git)
cd canva
npm install
npm start
```
